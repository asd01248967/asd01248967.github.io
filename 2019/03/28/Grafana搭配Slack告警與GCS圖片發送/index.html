Grafana搭配Slack告警與GCS圖片發送

呈上個主題，建立好dashboard與panel的分析圖表後(cpu、disk、ram、network等)

我們需要讓有一個機制可以在監控突然出現異常的時候可以發出通知告知管理人員

並且發送相對應的告警的截圖告知是哪一個監控告警，目前的值維多少，圖形是如何的趨勢

Configuration alert

step.1 首先先註冊一個slack的帳號與下載slack的應用程序或app

step.2 創建一個channel的頻道(也可以說是workspace的地方)，並將需要的管理人員都加入進去

stpe.3 前往slack的app Directory應用程序，登入後選擇所要發送告警通知的channel，去申請Incoming WebHooks的URL

網址：https://slack.com/apps

step.4 回來Grafana的Alerting → notifications channels → New Channel

填上名稱、類型、web hook url之後把send on all alerts與include image給勾選起來後點選save & test
